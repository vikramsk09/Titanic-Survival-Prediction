## 📌 Project Overview
This project is a Machine Learning-based Titanic Survival Prediction system built using Python. The model analyzes passenger data to predict whether a passenger survived the Titanic disaster.

It demonstrates a complete end-to-end classification workflow, including data preprocessing, visualization, model training, and evaluation.

---

## 🎯 Objective
- Predict passenger survival using machine learning
- Apply classification algorithms like Logistic Regression
- Gain hands-on experience with real-world historical datasets

---

## 🛠️ Technologies & Libraries Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## ⚙️ Machine Learning Workflow
1. **Data Collection**
   - Used the Titanic dataset containing passenger details

2. **Data Preprocessing**
   - Handling missing values
   - Encoding categorical variables
   - Feature selection and cleaning

3. **Data Visualization**
   - Used **Matplotlib** and **Seaborn** to analyze survival patterns

4. **Train-Test Split**
   - Used `train_test_split` to divide the dataset into training and testing sets

5. **Model Building**
   - Implemented **Logistic Regression** for classification

6. **Model Evaluation**
   - Evaluated performance using **accuracy_score**

---

## 📊 Features
- Predicts survival of Titanic passengers
- Uses Logistic Regression for classification
- Includes data visualization for insights
- Clean and structured ML pipeline
